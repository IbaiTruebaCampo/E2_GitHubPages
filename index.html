Presentación de aplicación CRUD

Compilador de datos para territorios

Funcionalidad:

Esta es una aplicación que nos permite hacer uso de una base de datos en Firebase creando diferentes campos de información, los cuales podremos editar o eliminar según nuestras necesidades.

Cómo furnciona:

Primero nos validamos en la aplicación para que nos conceda acceso a las diferentes funcionalidades. Para ello tendremos que estar registrados. El propio menú nos da la opción de acceder con nuestro correo o registrarnos. Una vez pulsemos el botón “Registrar” nos aparecerá una ventana en la que podremos introducir nuestro correo electrónico y una contraseña que tendrá que contener un mínimo de 6 caracteres. Una vez introducidos los datos se enviara un correo de validación para confirmar el registro el cual se almacenara en nuestro proyecto en Firebase. Una vez validados podremos volver a la página de la aplicación y refrescarla o bien si ya estábamos registrados introducir nuestros datos para dentro de los campos correspondientes. Una vez dentro, los botones de registro se sustituyen por un único botón para poder cerrar la sesión. Además se habrán cargado una serie de campos que tendremos que completar para poder guardar posteriormente en la base de datos de Firebase. Una vez en la base de datos visualizaremos la información introducida en una especie de tabla, en la que veremos los diferentes elementos que se crean al rellenar los campos las veces que deseemos. Estos mismos elementos los podremos manipular gracias a los botones que tenemos al final de ellos, ya sea bien para editarlos o eliminarlos. Una vez pulsemos para editarlos, los campos se rellenaran automáticamente con la información del elemento seleccionado y el botón de guardar cambiara por otro que nos permite editar, por lo que solo tendremos que cambiar los datos que deseemos y pulsar dicho botón. Si por el contrario lo que queremos es borrar datos solo tendremos que pulsar el botón correspondiente y confirmar la operación.
